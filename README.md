# Airbnb Clone — WanderLust

A full-stack Airbnb-like application deployed at:
https://delta-projects-gv9x.onrender.com

## Features

- Dynamic property listing page
- User signup and login
- Map feature for airbnb location
 

## Tech Stack
➤ Frontend
    •EJS
    •CSS
    •JAVASCRIPT
➤ Backend
   •EXPRESS
   •NODE.JS
   •MVC STRUCTURE


## Run Locally

1. Clone the repo  
   `git clone https://github.com/ABHI156-x/delta-projects.git`

2. Install deps  
   `npm install`

3. Start the server  
   `npm start`
